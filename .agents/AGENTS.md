# Project Rules

- **Page-Specific Changes**: Any requested changes (styles, layout, content) must be applied **only** to the specific page requested by the user. Do not make changes global or apply them to other pages across the website unless explicitly instructed to do so.
- **Page-Specific Images**: When adding or using images for a specific page, keep them within that page's specific folder or scope, and do not use them on other pages.
